HTML Nedir?

Anasayfa | Yazılar

HTML, web sayfalarının yapısını oluşturmak için kullanılan bir işaretleme dilidir.

HTML sayesinde başlıklar, paragraflar, listeler ve bağlantılar oluşturabiliriz.

Web geliştirmenin temelini HTML oluşturur ve genellikle CSS ve JavaScript ile birlikte kullanılır.
